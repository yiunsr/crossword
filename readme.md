## 프로그램 개발내용
* 한국어 단어를 이용한 십자말 풀이
* 개발환경
  * chrome과 firefox에서 사용 가능한 확장프로그램
  * extension.js + vue3 을 이용한 개발


## 게임 시나리오
* 게임 stage 가 있고 단계별로 난위도가 조금씩 증가함
* 맞추어야 할 단어
  * 고사성어, 요즘 뉴스 기사에 자주 나오는 말, 유명인 이름, 일상적인 어휘
* 하나의 stage 마다 테마가 있음
  * 과학, 국어, 시사
  * 하나의 테마로 stage 내 모든 단어로 구성할 수 없기 때문에 조금씩 다른 테마의 단어를 이용할 수도 있음
* 사용자가 추가한 단어로 게임에서 이용할 수 있어야 한다.
* 자주 틀리는 단어는 자주 등장해야 한다.
* 스테이지 마다 사용해야 하는 단어가 있고, 그 외에는 랜덤으로 추가 되는 단어가 있다.
  * 따라서 같은 stage 일지라도 게임 시작할 때 다른 십자말풀이로 구성될 수 있다.
  * 이 때 필수 사용단어는 꼭 추가되어야 한다.
* 따로 시간 제한은 없으나 게임진행 되는 시간은 표시된다.
* 단어에 대한 설명은 LLM으로 자동으로 생성된다.
  * 단어와 해당 설명은 미리 생성된다. xml 파일로 생성된다.
